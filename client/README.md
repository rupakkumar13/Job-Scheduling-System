# 🎨 Job Scheduling System - Client

This is the frontend component of the Job Scheduling System, built with **React** and **Vite**. It provides an interactive interface for users to manage jobs and visualize the optimal scheduling results generated by the backend.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- Backend server running (default: `http://localhost:5001`)

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## 💻 Features

- **Interactive Dashboard**: A sleek, two-column layout for job entry and results.
- **Dynamic Form**: Add jobs with validation for deadlines and profit values.
- **Live Updates**: The job list and schedule results update instantly without page reloads.
- **Modern UI/UX**:
  - Responsive design using CSS Grid and Flexbox.
  - Glassmorphism effects and vibrant color badges.
  - Interactive hover states and smooth transitions.
  - Status messages for user feedback (success/error).

## 🛠️ Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite 8
- **Styling**: Vanilla CSS (Custom properties/Variables)
- **API Communication**: Fetch API

## 📂 Structure

- `src/App.jsx`: Contains the main application logic, state management, and API calls.
- `src/index.css`: Custom design system including typography, color palette, and component styles.
- `public/`: Static assets and icons.

## 🔧 Configuration

The client communicates with the backend via the `API_BASE` constant defined in `App.jsx`. By default, it points to `http://localhost:5001`.
